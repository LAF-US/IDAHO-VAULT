## Admin

Organization setup, security, user management, and billing.

The Admin Panel is where you set up your organization, configure SSO, manage API keys and users, and track billing. Use it to control access, enforce security policies, and monitor usage across workspaces.

Quickstarts

Choose a quickstart to begin. Each guide walks you through a key admin task.

### [Create your organization](https://docs.mistral.ai/getting-started/quickstarts/admin/create-organization)

Set up billing, workspaces, and invite your first team members.

15 min to complete

### [Configure SSO](https://docs.mistral.ai/getting-started/quickstarts/admin/configure-sso)

Set up SAML Single Sign-On for enterprise authentication.

15 min to complete

### [Manage workspaces & API keys](https://docs.mistral.ai/getting-started/quickstarts/admin/manage-workspaces)

Create isolated workspaces, generate keys, and set usage limits.

10 min to complete

Explore

Configure SSO, manage API keys, assign roles and permissions, and track usage and billing across workspaces.

### [Back Office](https://docs.mistral.ai/admin/security-access/back-office)

View your organization dashboard, manage workspaces, and configure workspace settings.

### [SSO & Authentication](https://docs.mistral.ai/admin/security-access/sso)

Set up SAML-based single sign-on and enforce identity provider authentication for your organization.

### [API Keys](https://docs.mistral.ai/admin/security-access/api-keys)

Create, rotate, and revoke API keys across workspaces.

### [User Management](https://docs.mistral.ai/admin/user-management-finops/user-management)

Invite members, assign roles, and control permissions across your organization.

### [Billing & Usage](https://docs.mistral.ai/admin/user-management-finops/billing)

Track consumption, manage subscriptions, and set usage limits per workspace.
