# True Name - Explorer Companion

This note accompanies `- Fandom, Inc. - True name.md`.

## What The Note Seems To Do

The source note treats identity as a force-bearing surface.

A true name is not merely descriptive. It is operative.

Humans can bury birth names beneath assumed ones. Demons can move under
bynames. But the deeper name still matters, because it changes what another
being can do to you and what you can resist in return.

## Why It Matters Here

The House has already been warning about naming.

`WHO.md` did not behave like a missing biography. It behaved like a dangerous
identity surface under pressure.

This Bartimaeus note clarifies why. A registry name, a role name, a working
name, and a true name may not be the same thing. To collapse them too quickly
is not precision. It is exposure.

## Field Note

The lesson here is not that every hidden name should be dragged into the open.

It is that names require custody.

Sometimes witness means recording the alias that a chamber can safely bear.
Sometimes it means refusing to force the deeper name until a neighboring record
authorizes it.

That makes this note a sharp companion to the `WHO` cluster:
not because it answers `who`,
but because it explains why the question can bite.
