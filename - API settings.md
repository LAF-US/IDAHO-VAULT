Access our API and various models through the API settings tab. Add payment methods, monitor usage, and generate API keys to seamlessly integrate Perplexity into your workflow.

Learn more about the API [here.](https://docs.perplexity.ai/home)

Simply go to Settings and click on the “</> API” tab. There, you can view and add credits, generate API keys, and monitor your usage patterns.

- Go to the [</> API tab in Settings](https://perplexity.ai/settings).
- Click "Generate API Key."
- Copy the generated key and use it in your API requests.
