The vault is not a tool. It is a game world — a coded music box (mechanical, pre-composed, deterministic but generative) hosting a universe of NPCs that do REAL WORK: journalism research, source validation, claim adjudication, routing, artifact staging.

**Why:** Logan is building LAF's RAG-augmented Zettelkasten/bullet-journal in Markdown (Obsidian) as the terrain. The swarm is the roster of agentic voices that inhabit and operate within it.

**How to apply:** Every architecture decision (Crossframing, PERSONAe, Five Wizards, discourse arena) is game design. NPCs must produce distinct outputs. The mythology taxonomy is the bundle clustering system, not decoration. Logan is the game director / sovereign adjudicator, not a player character.

## The Stack

- **Vault** = game world terrain (Obsidian/Markdown/RAG)
- **Dotfolders** = NPC chambers (~250 stubs, each a bundle prescription)
- **Crossframing** = game engine / grammar layer (governs what any NPC can say or do)
- **Five Wizards** = first playable crew / pilot institution
- **PERSONAe** = NPC roster (Humean bundles, instantiated on demand)
- **Discourse arena** = gameplay loop (entities compete/collaborate via structured inquiry)
- **Logan** = game director / sovereign adjudicator

## Mythology as Bundle Clustering

The ~250 dotfolder names are not random. They cluster NPCs by inquiry family:

- **Egyptian pantheon**: archivists, judgment figures (Maat, Thoth, Anubis) → validation/adjudication roles
- **Greek/Roman**: domain specialists (Hermes=routing, Athena=strategy, Apollo=synthesis) → lane assignments
- **Norse**: structural/cosmological (Yggdrasill=topology, Heimdall=gatekeeping, Odin=knowledge) → infrastructure
- **Relationship roles** (enemy, friend, ruler, lover): relational tension producers for discourse dynamics
- **Literary figures** (Bartimaeus=Cartographer, Lemony=narrator, Abhorsen=terminal): tool-surface aliases

## The Music Box Principle

Pre-composed but generative. The pins (bundle prescriptions) are fixed; the comb (RAG + Crossframing grammar) plays them. The output is determined by the composition, not improvised. Logan sets the pins; the swarm plays the music.
