# Taskbone

Taskbone is an Obsidian plugin that provides OCR (optical character recognition) capabilities and optional Todoist task sync. It can extract text from images pasted into notes.

## Getting Started

### Install the Plugin

1. Open Obsidian **Settings**
2. Go to **Community plugins**
3. Enable community plugins if not already on
4. Browse and search for **"Taskbone"** — may appear under its old name "Taskbone OCR"; install it
5. **Enable** the plugin

### Connect to Taskbone

After enabling, you will be prompted to connect to Taskbone. Click the link and login or register using:
- Your **Todoist account**, or
- An **email address**

### Account Type and Available Commands

| Registration method | OCR commands | Todoist sync commands |
| ------------------- | ------------ | --------------------- |
| Todoist account     | ✅            | ✅                     |
| Email only          | ✅            | ❌                     |

If you registered by email, only the OCR commands will be available — Todoist sync requires a connected Todoist account.
