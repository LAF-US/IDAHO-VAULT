## Summary

Linear advertises deep integrations so it can act as the single source of truth for product work. Core connectors include Slack for creating issues from messages and syncing threads, GitHub/GitLab for PR/commit automation with two-way issue sync, and AI agents as teammates. An integration directory lists 150+ additional options, and a GraphQL API powers custom flows to keep data in sync and eliminate manual updates between tools.

## Key Integrations (from issue)

- Slack — Create issues from Slack messages and keep threads synced.
- GitHub / GitLab — Automate pull request and commit workflows; keep issues in sync both ways.
- Agents — Deploy AI agents that work alongside humans.
- Integration directory — 150+ supported tools (Intercom, Zendesk, Figma, etc.).
- Linear API — GraphQL API for custom integrations; see developer docs.
