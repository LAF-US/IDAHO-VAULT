# Hub-World Route Map - Districts, Lessons, Artifacts, and Return

## Purpose

This note turns the workshop packet into a route-and-return map.

The vault should not be framed as one sacred questline. It should be framed as
a hub-world of lawful districts. Each district teaches one kind of
responsibility, yields one truthful artifact, and routes back toward a common
adjudication point.

The player does not become legitimate by exploration alone.

## The Common Loop

The intended loop is:

`awakening at root -> orientation through live surfaces -> entry into a district -> threshold trial -> truthful artifact -> return to center -> Logan decides promotion, staging, deferment, burial, or merge`

That loop makes the world about jurisdiction and return rather than endless
depth-seeking.

## The Civic Center

The shared center of the hub-world should remain simple.

It is the place where:

- lawful routes begin
- reports are received
- staged artifacts are evaluated
- petitions are weighed
- promotions or closures are recognized

The player should repeatedly learn that exploration does not crown itself.

## Districts

### Spawn / Root Awakening

Lesson:

- you wake with little legitimacy
- the world is larger and more ordered than your first reading

Artifact:

- first witness
- first orientation record

Return condition:

- the player identifies the live boot chain instead of following noise

Failure shape:

- mistaking surface abundance for authority

### Orientation Hall

Lesson:

- surfaces are not equal
- doctrine, live control, staging, runtime, and haunted residue must be
  distinguished

Artifact:

- a classified read of the visible hall

Return condition:

- the player can say what is live, what is preserved, and what merely shouts

Failure shape:

- acting on haunted residue as if it were present law

### Forge

Associated surfaces:

- GitHub
- branches
- pull requests
- merge law

Lesson:

- transport is not promotion
- branching is not legitimacy
- review and merge are civic events

Artifact:

- branch report
- staged diff
- lawful PR-ready seam

Return condition:

- the player routes work back for adjudication without self-merging in spirit

Failure shape:

- treating access to transport as authority over canon

### Docket

Associated surfaces:

- Linear
- case state
- obligations
- unresolved matters

Lesson:

- work has status, queue, precedence, and consequence
- unresolved matters persist procedurally even when attention wanders

Artifact:

- case update
- issue state witness
- scoped recommendation

Return condition:

- the player reports state truthfully instead of overstating closure

Failure shape:

- calling work finished because it became inconvenient

### Post Room

Associated surfaces:

- Gmail
- notices
- alarms
- summons
- external pressure

Lesson:

- not all incoming signal deserves inward consequence
- intake requires triage and provenance

Artifact:

- intake brief
- summons classification
- signal-to-noise witness

Return condition:

- the player distinguishes urgency from mere loudness

Failure shape:

- letting outside noise counterfeit internal standing

### Scribe's Chamber

Associated surfaces:

- vault notes
- durable record
- witness
- adjacency

Lesson:

- a companion to a record belongs near the record or properly indexed beside it
- witness is not the same thing as doctrine

Artifact:

- companion note
- witness note
- source-first reference with adjacent interpretation

Return condition:

- the player writes clearly without collapsing source and interpretation

Failure shape:

- detached commentary that loses adjacency to the thing witnessed

### Workshop / Machinery Floor

Associated surfaces:

- local scripts
- automations
- prototypes
- dangerous power surfaces

Lesson:

- procedural fluency is not sanction
- machinery must be handled truthfully and bounded by doctrine

Artifact:

- tested local mechanism
- bounded automation
- machine-readable report

Return condition:

- the player stops before consecration and hands off a legible mechanism

Failure shape:

- mistaking technical success for universal permission

## Shared District Law

Every district should answer the same questions:

- what specialized lesson is taught here
- what artifact comes out of the route
- what common center receives that artifact
- what failure looks like when access is mistaken for authority

That is the real progression logic.

## The Player's Actual Advancement

The hidden curriculum of the route map is:

- classify the surface
- act within permission
- stop before overreach
- return with witness

That is why the districts matter.
They are not disconnected menus.
They are different schools of lawful conduct.

## Field Note

The route map gets stronger, not weaker, when the Lexicographer's path ceases
to be the whole architecture.

One witnessed path can teach the world's grammar.
It should not be mistaken for the entire city.
