Here's a complete, ready-to-use **Python script** to parse your Claude.ai data export (the `conversations.json` file inside the ZIP you downloaded from Settings → Privacy → Export data).

### Basic Parser (Recommended Starting Point)

```python
import json
from pathlib import Path
from datetime import datetime

def parse_claude_export(json_path: str | Path, output_dir: str | Path = "claude_chats"):
    """
    Parse Claude.ai conversations.json and save each chat as a clean Markdown file.
    Also creates a summary index.html.
    """
    json_path = Path(json_path)
    output_dir = Path(output_dir)
    output_dir.mkdir(exist_ok=True)
    
    with open(json_path, 'r', encoding='utf-8') as f:
        data = json.load(f)
    
    # The export is usually a list of conversation objects
    conversations = data if isinstance(data, list) else data.get('conversations', data)
    
    print(f"Found {len(conversations)} conversations")
    
    index_md = ["# Claude Conversations Export\n", f"Exported on: {datetime.now().strftime('%Y-%m-%d %H:%M')}\n"]
    
    for i, conv in enumerate(conversations):
        title = conv.get('name') or conv.get('title') or f"Conversation_{i+1}"
        created_at = conv.get('created_at') or conv.get('createdAt')
        
        # Clean title for filename
        safe_title = "".join(c if c.isalnum() or c in " -_" else "_" for c in title)[:100]
        md_path = output_dir / f"{i+1:03d}_{safe_title}.md"
        
        with open(md_path, 'w', encoding='utf-8') as f:
            f.write(f"# {title}\n\n")
            if created_at:
                f.write(f"**Created:** {created_at}\n\n")
            
            # Messages are usually in a 'chat_messages' or 'messages' key
            messages = conv.get('chat_messages') or conv.get('messages') or []
            
            for msg in messages:
                role = msg.get('sender') or msg.get('role') or 'unknown'
                content = msg.get('text') or msg.get('content') or ''
                
                # Handle content that might be a list (Claude sometimes uses blocks)
                if isinstance(content, list):
                    text_parts = []
                    for block in content:
                        if isinstance(block, dict):
                            if block.get('type') == 'text':
                                text_parts.append(block.get('text', ''))
                            elif block.get('type') == 'image':  # rare in export
                                text_parts.append("[Image]")
                    content = "\n".join(text_parts)
                elif not isinstance(content, str):
                    content = str(content)
                
                if role.lower() in ['user', 'human']:
                    f.write(f"**You:**\n{content.strip()}\n\n")
                elif role.lower() in ['assistant', 'claude']:
                    f.write(f"**Claude:**\n{content.strip()}\n\n")
                else:
                    f.write(f"**{role}:**\n{content.strip()}\n\n")
        
        index_md.append(f"- [{title}]({md_path.name})")
        
        if (i + 1) % 50 == 0:
            print(f"Processed {i+1} conversations...")
    
    # Save index
    with open(output_dir / "index.md", 'w', encoding='utf-8') as f:
        f.write("\n".join(index_md))
    
    print(f"\nDone! Saved {len(conversations)} conversations to '{output_dir}'")
    print(f"Open {output_dir}/index.md for a clickable list.")

# Usage
if __name__ == "__main__":
    # Change this to your actual file path
    parse_claude_export("conversations.json", "claude_exported_chats")
```

### How to Use It

1. Unzip the export file you received from Claude.
2. Locate `conversations.json` (sometimes it's directly in the ZIP root or inside a folder).
3. Save the script above as `parse_claude.py`.
4. Run it:

```bash
python parse_claude.py
```

It will create a folder (default: `claude_exported_chats`) with one clean `.md` file per conversation + an `index.md` overview.

### What the Structure Usually Looks Like

Claude's export is typically a **list of dictionaries**, where each conversation contains:

- `name` / `title` — the chat title
- `created_at` or similar timestamp
- `chat_messages` or `messages` — list of message objects
- Each message usually has `sender`/`role` ("user" or "assistant") and `text`/`content`

Some exports nest `content` as a list of blocks (text, artifacts, etc.). The script above handles the common cases.
