## API Reference

## Popular tasks[Chat Completion](https://huggingface.co/docs/inference-providers/tasks/chat-completion)

[

Generate a response given a list of messages in a conversational context.

](https://huggingface.co/docs/inference-providers/tasks/chat-completion)[

Feature Extraction

Converting a text into a vector often called "embedding".

](https://huggingface.co/docs/inference-providers/tasks/feature-extraction)[

Text to Image

Generate an image based on a given text prompt.

](https://huggingface.co/docs/inference-providers/tasks/text-to-image)[

Text to Video

Generate an video based on a given text prompt.

](https://huggingface.co/docs/inference-providers/tasks/text-to-video)

## Other tasks[Audio Classification](https://huggingface.co/docs/inference-providers/tasks/audio-classification)

[

Audio classification is the task of assigning a label or class to a given audio.

](https://huggingface.co/docs/inference-providers/tasks/audio-classification)[

Automatic Speech Recognition

Automatic Speech Recognition (ASR), also known as Speech to Text (STT), is the task of transcribing a given audio to text.

](https://huggingface.co/docs/inference-providers/tasks/automatic-speech-recognition)[

Fill Mask

Mask filling is the task of predicting the right word (token to be precise) in the middle of a sequence.

](https://huggingface.co/docs/inference-providers/tasks/fill-mask)[

Image Classification

Image classification is the task of assigning a label or class to an entire image. Images are expected to have only one class for each image.

](https://huggingface.co/docs/inference-providers/tasks/image-classification)[

Image Segmentation

Image Segmentation divides an image into segments where each pixel in the image is mapped to an object.

](https://huggingface.co/docs/inference-providers/tasks/image-segmentation)[

Image to Image

Image-to-image is the task of transforming a source image to match the characteristics of a target image or a target image domain.

](https://huggingface.co/docs/inference-providers/tasks/image-to-image)[

Object Detection

Object Detection models allow users to identify objects of certain defined classes.

](https://huggingface.co/docs/inference-providers/tasks/object-detection)[

Question Answering

Question Answering models can retrieve the answer to a question from a given text, which is useful for searching for an answer in a document.

](https://huggingface.co/docs/inference-providers/tasks/question-answering)[

Summarization

Summarization is the task of producing a shorter version of a document while preserving its important information.

](https://huggingface.co/docs/inference-providers/tasks/summarization)[

Table Question Answering

Table Question Answering (Table QA) is the answering a question about an information on a given table.

](https://huggingface.co/docs/inference-providers/tasks/table-question-answering)[

Text Classification

Text Classification is the task of assigning a label or class to a given text.

](https://huggingface.co/docs/inference-providers/tasks/text-classification)[

Text Generation

Generate text based on a prompt.

](https://huggingface.co/docs/inference-providers/tasks/text-generation)[

Token Classification

Token classification is a task in which a label is assigned to some tokens in a text.

](https://huggingface.co/docs/inference-providers/tasks/token-classification)[

Translation

Translation is the task of converting text from one language to another.

](https://huggingface.co/docs/inference-providers/tasks/translation)[

Zero Shot Classification

Zero shot classification is the task to classify text without specific training for the task.

](https://huggingface.co/docs/inference-providers/tasks/zero-shot-classification)
